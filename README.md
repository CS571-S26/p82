# CS571 Web Project - Badger Gameday Survival Guide
**Project Description**

The Badger Gameday Survival Guide is a web app designed to alleviate
the chaos of a UW-Madison home football game. If you’ve ever been to
a home game, you have probably experienced the shared frustrations
of finding a place to park (that is somewhat within walking distance of
Camp Randall), and finding a place to eat or grab a drink that doesn’t
have a line out the door. This app focuses on improving these issues to
optimize users’ pre-game and post-game experience. 
##
**Details**

The app will be built using React & Vite and styled with React-Bootstrap to ensure a
responsive, accessible, and professional UI. Using react router, the site
will contain 5 primary pages: a homepage, about page, live parking map, restaurant finder,
and gameday essentials dashboard. The “live parking map” will
integrate the City of Madison’s Open Data API to display real-time
availability for city-owned ramps, combined with an embedded,
interactive map for navigation/distance calculation through Google
Maps API. The “restaurant finder” will use the Google Places API (or a
different free open-source API) to help users locate available dining
options near their current location. The main interactive elements will
allow users to select a parking lot and a destination (Camp Randall or
the restaurant) to calculate a walking duration and route, and allow
users to select a restaurant and place a reservation (if they take
reservations). The app will also consist of at least 12 React components,
including navigation bars, interactive map pins, and filterable data
cards. 
##
**Future Improvements**

By the end of the semester, I am hoping to also incorporate an AI
agent to provide personalized itineraries based on user preferences for
food, parking prices/locations, and walking distances.
